### Операции / операторы
```
* Унарная операция - операция выполняется над одним объектом (операндом);
* Бинарная - ... над двумя операндами;
* Тернарная - ... над тремя операндами.
* Тернарный оператор: [первый операнд - условие] ? [второй операнд if_true] : [третий операнд if_false]

* "=" - оператор присваивания;
* "==" - равно;
* ">", ">=" - больше, больше или равно;
* "<", "<=" - меньше, меньше или равно;
* "!=" - не равно (бинарный);
* "!" - не (унарный);
* "&" - сокращенное "и" (если одна часть условия не проходит, то вторая часть не проверяется);
* "&&" - "и" (в любом случае проверятся обе части условия);
* "|" - сокращенное "или" (если одна часть условия не проходит, то вторая часть не проверяется);
* "||" - "или" (в любом случае проверятся обе части условия);
* "??" - оператор NULL-объединения (str ?? "нет данных" - в случае NULL в str, в консоли выведет сообщение "нет данных");
* "??=" - оператор присваивания для значения NULL;
* "?." - оператор условного NULL (для проверки на NULL);

* ref - для передачи параметра по ссылке (изменится значение после метода);

* out - то же, что и ref, отличие в том, что с out в методе нужно задать значение для аргумента;

* in - для считывания данных (read only);

* params - считывает данные через запятую для массива, например;

* void Resize<T>(ref T[] array, int size) - <T> для того, чтобы работать с любым типом данных;

* "break" - используется для прерывания цикла, когда нужный элемент найден;

* new Random().Next("от какого-то числа", "до какого-то числа") - Оператор для рандомных чисел;

* String.Empty - пустая строка;

* Console.Clear(); - очистка консоли;

* int <name> = <array_name>.Max() or array.Min() or array.Sum() - to find max or min value or sum of the values of the array;

* int <name> = <array_name>.Where(i => i % 2 == 0).Sum() - to find sum of the even numbers;

* int <name> = <array_name>.Where(i => i < 0).FirstOrDefault() - to find first number in the array that is less than 0; 

* int[] <array_name> = <array_name>.Distinct().ToArray() - to leave only unique values (remove repeated values);

* int[] <array_name> = <array_name>.OrderBy(i => i).ToArray() or array.OrderByDescending(i => i).ToArray() - to sort values by ascending (отсортировать по возрастанию) or descending (убыванию);

* Array.Sort(<array name>) - to sort values by ascending;

* Array.Reverse(<array name>) - to reverse the array;

* int <name> = Array.Find or FindLast(<array name>, i => i < 70) - to find first or last number in the array that is less than 70;

* int[] <name> = Array.FindAll(<array name>, i => i < 70) - to find all numbers in the array that are less than 70;

* int <name> = Array.FindIndex or FindLastIndex(<array name>, i => i == 77) - to find index of the first or last number 77 in the array;

* break - прерывание цикла;

* continue - переход на начало цикла (прерывание текущей итерации цикла);

* \t - табуляция;

* Enumerable.Repeat(<number>, <qty to repeat>).ToArray() - заполнить массив определенным значением;

* Enumerable.Range(<starting number>, <qty to fill>) - ToArray() - заполнить массив начиная с определенного значения в нарастающем порядке;

* Console.SetCursorPosition(<отступ слева>, <отступ сверху>);

* Convert.To<тип данных> - преобразование из одного типа данных в другой;

* <тип данных>.Parse(<тип данных string>) или <тип данных>.TryParse(<тип данных string>, out <тип данных>) - преобразование из строки в другой числовой тип данных; TryParse может использоваться как BOOL;

* ConsoleKey <назв.переменной> = Console.ReadKey().Key; - для считывания знака с клавиатуры при нажатии на неё;

* switch (<тип данных_название переменной>)
{
    case 1:
        <действие>;
        break;
    case 2:
    ...
    default:
        <действие>;
        break;    
}
```

### Циклы
```
* for (int i = 0; i < length; i++)
{
    <действие, которое должно выполняться, пока цикл выполняется>
}

* "обратный for": for (int i = <начало отсчета>; i > конец отсчета; i--) 
{
    <действие, которое должно выполняться, пока цикл выполняется>
}
(можно работать с двумя переменными в цикле for);

* while(<условие> true)
{
    <действие, которое должно выполняться, пока цикл выполняется>
}

* do
{
    <действие, которое должно выполняться, пока цикл выполняется>
}
while(<условие> true)
```

#### Типы С#
```
Ссылочные типы - размещаются в управляемой куче (class);
Значимые типы - размещаются в стеке (struct, enum);
```

```
Оперативная память:
* STACK - для хранения переменных, для вызова методов (дефолтная выделяемая память - 1 МБ). Мало, но быстро.

* HEAP (управляемая куча) - исп-ся по надобности (мб выделено до 8 ТБ для х64 системы). Много, но медленно.
```


#### Различие между *while* и *do while* в том, что в цикле *while* сначала проверяется условие, а в цикле *do while* - сначала выполняется действие, потом условие.

### Типы данных
```
* int - целочисленный (От -2 147 483 648 до 2 147 483 647);
```

### Функция (метод)
```
Пример написания метода:

int Max(int arg1, int arg2, int arg3)
{
    int result = arg1;
    if(arg2 > result) result = arg2;
    if(arg3 > result) result = arg3;
    return result;
}
```

```
/// - для добавления описания к методу
```

```
Кортеж - когда в одну функцию можно добавить несколько аргументов. Пример:

(int, int) GetSumms(int[] array)
{
    int positiveSumm = 0;
    int negativeSumm = 0;

    for (int i = 0; i < array.Length; i++)
    {
        if(array[i] < 0)
            negativeSumm += array[i];
        else
            positiveSumm += array[i];
    }
    return (positiveSumm, negativeSumm);
}
```

```
Метод для задания считывания определенного знака в числах с дробями:

using System.Globalization;

NumberFormatInfo numberFormatInfo = new NumberFormatInfo()
{
    NumberDecimalSeparator = ".",
};
<тип данных> <название переменной> = Convert.ToDouble(<тип данных считывания строки>, numberFormatInfo);
```

### Массивы
```
Одномерные массивы:

<ТИП ДАННЫХ>[] <ИМЯ> ={Значение1,Значение2,...}
int[] array = {9, 28, 1, 32, 1900, ...}
int[] array = new int[5];
int[] array = new int[]{0, 0, 0, 0, 0}
int[] array = new int[5]{1, 0, 2, 0, 3}
```
#### Обращение к массиву: array[index]
#

```
Двумерные массивы:

<ТИП ДАННЫХ>[,] <ИМЯ> = new <ТИП ДАННЫХ>[<кол-во строк>, <кол-во столбцов>];

#### Обращение к массиву: <ИМЯ>[<кол-во строк>, <кол-во столбцов>];
#### Для задания длины строк массива: <ИМЯ>.GetLength(0);
#### Для задания длины столбцов массива: <ИМЯ>.GetLength(1);
```

```
Зубчатый массив:

<ТИП ДАННЫХ>[][] <ИМЯ> = new <ТИП ДАННЫХ>[<кол-во элементов массива>][] - двумерный массив;
<ТИП ДАННЫХ>[][][] <ИМЯ> = new <ТИП ДАННЫХ>[<кол-во элементов массива>][][] - трехмерный массив;
<ИМЯ>[<индекс элемента>] = new <ТИП ДАННЫХ>[<кол-во элементов массива>];
...
<ИМЯ>[<индекс элемента>] = new <ТИП ДАННЫХ>[<кол-во элементов массива>];

Обращение к массиву: <ИМЯ>[<индекс элемента>][<индекс элемента>];
```
