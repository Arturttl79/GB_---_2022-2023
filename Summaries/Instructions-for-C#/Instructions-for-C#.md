### Разные операторы
```
* "=" - оператор присваивания;
* "==" - равно;
* ">", ">=" - больше, больше или равно;
* "<", "<=" - меньше, меньше или равно;
* "!=" - не равно (бинарный);
* "!" - не (унарный);
* "&" - сокращенное "и" (если одна часть условия не проходит, то вторая часть не проверяется);
* "&&" - "и" (в любом случае проверятся обе части условия);
* "|" - сокращенное "или" (если одна часть условия не проходит, то вторая часть не проверяется);
* "||" - "или" (в любом случае проверятся обе части условия);
* "break" - используется для прерывания цикла, когда нужный элемент найден;

* new Random().Next("от какого-то числа", "до какого-то числа") - Оператор для рандомных чисел;

* String.Empty - пустая строка;

* Console.Clear(); - очистка консоли;

* continue - переход на начало цикла;

* Convert.To<тип данных> - преобразование из одного типа данных в другой;

* <тип данных>.Parse(<тип данных string>) или <тип данных>.TryParse(<тип данных string>, out <тип данных>) - преобразование из строки в другой числовой тип данных; TryParse может использоваться как BOOL;

* ConsoleKey <назв.переменной> = Console.ReadKey().Key; - для считывания знака с клавиатуры при нажатии на неё;

* switch (<тип данных_название переменной>)
{
    case 1:
        <действие>;
        break;
    case 2:
    ...
    default:
        <действие>;
        break;    
}
```

### Циклы
```
* for (int i = 0; i < length; i++)
{
    <действие, которое должно выполняться, пока цикл выполняется>
}

* "обратный for": for (int i = <начало отсчета>; i > конец отсчета; i--) 
{
    <действие, которое должно выполняться, пока цикл выполняется>
}
(можно работать с двумя переменными в цикле for);

* while(<условие> true)
{
    <действие, которое должно выполняться, пока цикл выполняется>
}

* do
{
    <действие, которое должно выполняться, пока цикл выполняется>
}
while(<условие> true)
```
#### Различие между *while* и *do while* в том, что в цикле *while* сначала проверяется условие, а в цикле *do while* - сначала выполняется действие, потом условие.

### Типы данных
```
* int - целочисленный (От -2 147 483 648 до 2 147 483 647);
* long - целочисленный (Oт -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807);
* float - с плавающей запятой (Точность - 6–9 цифр);

* decimal - с плавающей запятой (Точность - 28-29 знаков);

* string - символьный тип, текст как последовательность значений *char*;
* bool - логическое значение: true, false (по умолчанию);
* object - особый тип, прародитель всех верхних типов данных;
* dynamic - особый тип, может меняться по ходу программы;
```

### Функция (метод)
```
Пример написания метода:

int Max(int arg1, int arg2, int arg3)
{
    int result = arg1;
    if(arg2 > result) result = arg2;
    if(arg3 > result) result = arg3;
    return result;
}
```

```
Кортеж - когда в одну функцию можно добавить несколько аргументов. Пример:

(int, int) GetSumms(int[] array)
{
    int positiveSumm = 0;
    int negativeSumm = 0;

    for (int i = 0; i < array.Length; i++)
    {
        if(array[i] < 0)
            negativeSumm += array[i];
        else
            positiveSumm += array[i];
    }
    return (positiveSumm, negativeSumm);
}
```

```
Метод для задания считывания определенного знака в числах с дробями:

using System.Globalization;

NumberFormatInfo numberFormatInfo = new NumberFormatInfo()
{
    NumberDecimalSeparator = ".",
};
<тип данных> <название переменной> = Convert.ToDouble(<тип данных считывания строки>, numberFormatInfo);
```

### Массивы
```
Одномерные массивы:

#

```
Двумерные массивы:

<ТИП ДАННЫХ>[,] <ИМЯ> = new <ТИП ДАННЫХ>[<кол-во строк>, <кол-во столбцов>];

#### Обращение к массиву: <ИМЯ>[<кол-во строк>, <кол-во столбцов>];
#### Для задания длины строк массива: <ИМЯ>.GetLength(0);
#### Для задания длины столбцов массива: <ИМЯ>.GetLength(1);
